# ウサギ王国
はねつきとうかのwebサイト

# development set up

```shell
yarn
yarn dev
# > Local: http://localhost:3000/
# > Network: use `--host` to expose
```

# deploy
```shell
npm install -g firebase-tools
firebase login
# logged in manage account for Google

yarn deploy
```
